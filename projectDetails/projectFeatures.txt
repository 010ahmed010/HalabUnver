
================================================================================
  PROJECT BLUEPRINT: HALABUNVER – ALEPPO UNIVERSITY ECOSYSTEM (v2.0 - 2026)
================================================================================

1. CORE STRATEGY & VALUE PROPOSITION
--------------------------------------------------------------------------------
* PRIMARY GOAL: Build a zero-inventory academic & economic hub for 100k+ users.
* THE FOUR PILLARS:
  - HALAB ACADEMY: Two-tier YouTube model (Curated + Original Video Tutorials).
  - HALAB LIBRARY: Digital hub for Free/Paid Noutat & PDFs (Google Drive Hosted).
  - HALAB STORE: Physical Brokerage for student gear, hardware, and tools.
  - HALAB TALENT: Subscription-based Marketplace for student freelancers.
* RETENTION TOOL: Integrated GPA Tracker to drive daily recurring traffic.

2. THE REVENUE & BROKERAGE ENGINE
--------------------------------------------------------------------------------
* AD-GATE SYSTEM: Mandatory 10s reward-based view to unlock Academy/Library links.
* BROKERAGE STORE: Affiliate-style display of local hardware with delivery logic.
* SAAS SUBSCRIPTIONS: Monthly fees for "Talent" profiles and premium freelancer spots.
* LOCAL BANNERS: Targeted ad space for Aleppo-based training and service centers.

3. TECHNICAL ARCHITECTURE (OPTIMIZED FOR $6 VPS)
--------------------------------------------------------------------------------
* INFRASTRUCTURE: Dockerized Node.js Microservices (4 Independent Containers).
* GATEWAY: Nginx Reverse Proxy with 1-second Micro-caching (Drops CPU load 90%).
* DATABASE: MongoDB Atlas (Single Cluster) using Mongoose Pool (maxSize: 5).
* RESOURCE LIMITS: Strict Cgroup Hard Caps (512MB RAM per service) to prevent OOM.
* BANDWIDTH: 100% Offloaded to YouTube, Google Drive, and Cloudflare CDN.

4. PERFORMANCE & SCALABILITY
--------------------------------------------------------------------------------
* CONCURRENCY: Path-based routing (/api/v1/*) for high-speed request handling.
* MEMORY EFFICIENCY: Mongoose .lean() queries + Stateless JWT Authentication.
* FRONTEND: React + Vite 8 + Tailwind CSS v4 (Global CDN caching via Cloudflare).
* TOTAL ESTIMATED OPEX: ~$6.00 / Month.

5. IMPLEMENTATION ROADMAP
--------------------------------------------------------------------------------
* PHASE 1: "Academy & Library" Launch (Content-first to capture the 100k audience).
* PHASE 2: "Talent Marketplace" (SaaS subscriptions to generate immediate cash flow).
* PHASE 3: "Physical Store" (Expansion into local tools and hardware logistics).
